## Installation

- Install the development version from GitHub:

``` r
# install.packages("remotes")
remotes::install_github("umr1283/RefFreeEWAS")
```

- Install a particular version:

``` r
# install.packages("remotes")
remotes::install_github("umr1283/RefFreeEWAS@v3.0.0")
```
